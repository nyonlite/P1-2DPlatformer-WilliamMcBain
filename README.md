I chose to use the rigidbody2D velocity editing method of physics, because I knew a method to add acceleration and deceleration using this method.
I chose to use the Overlap Circle ground detection method because I needed few lines of script, and have found it to be an accurate method that is hard to find mistakes with.
I chose to implement both the Coyote Time and Input Buffer, since I had already implemented them in another project and was confident in using both.
